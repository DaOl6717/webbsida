Storage Area

A food stock management system built using an old Raspberry Pi and a barcode scanner. Connected to Home Assistant which enables displaying information on the HA dashboard at home, and automated shopping lists on my phone. This project is nearly done, but during testing, I noticed that I didn't use it that much. The system has since brought offline and the Raspberry is repurposed for image recognition in another upcoming project.

The Challenge

When I moved apartments, my new apartment included a small storage area in the actual apartment. To make the most of it, I put up a big shelf and started storing cans of food. However, it was difficult to know what I had in stock and when it expired, which became the start of this project.

Technical Details

An old Raspberry Pi 2 model B running running linux. Connected to it is an incompatible (but working) display for text input, a USB WiFi module and a barcode scanner. Through WiFi, it can locally connect to Home Assistant, which runs a MariaDB SQL database, storing information about the stored food. The information is then displayed on the Home Assistant dashboard.
