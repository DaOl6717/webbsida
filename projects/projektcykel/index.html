Projektcykel

"Projektcykel" or "Project Bike" in swedish, is a restoration and development project started in 2023 together with a friend. There isn't really a set out goal, we just want to add as much tech as possible to the bike.

[Placeholder]

The Challenge

Restoring the bike, understanding embedded systems, programming sensors, etc. There's a lot of challenges in this project, but the biggest one is probably finding the time to work on it...

Technical Details

Currently it consists of a bike, an Arduino Nano and a hall effect sensor. There isn't really much to it as of right now.
